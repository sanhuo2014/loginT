# 登录功能的 Node.js 应用程序

这个项目是一个基于 Node.js 的 web 应用程序，实现了用户登录的基本功能。程序中，我们使用了 express 框架来处理 HTTP 请求，对用户的用户名和密码进行了安全的处理和验证。

## 如何运行这个程序

首先，你需要安装 Node.js 和 npm。然后在项目目录中运行下列命令来安装依赖项：

```shell
npm install
```

安装完成依赖项后，使用下列命令启动应用程序：

```shell
node app.js
```

打开浏览器并访问 `localhost:3000`，你就可以看到登录页面了。输入用户名和密码，如果验证通过，你将被重定向到欢迎页面。
